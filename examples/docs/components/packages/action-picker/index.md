### API

#### ActionPicker
| 参数      | 说明             | 类型          | 默认值      | 是否必须 | 备注 |
|:----------|:-----------------|:--------------|:------------|:-----------------------------|:----
| leftList | 需传入一级菜单数据 | Array |                 | 否| 
| rightList | 需传入二级菜单数据 | Array |                 | 否| 


#### ActionPicker Events
| 事件     | 说明             | 参数类型          | 默认值      | 是否必须 | 备注 |
|:----------|:-----------------|:--------------|:------------|:-----------------------------|:----
| result | 返回选中的数据 | Object |           -      |-|-|-|
| fetchRightList | 自定义数据源时返回二级菜单的id | String |                -|-|-|-|
