### API

#### ActionTip
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| title | 标题 | String  |   | title |
| customClass | 弹窗class名 | String |   |   |

#### ActionTip Events
| 事件名      | 说明          | 返回值                      |
|---------- |-------------- |--------------------------- |
| clickMask | 点击蒙层触发的事件 |   | 

#### Slot
| name      | 说明          | 可选值                          | 默认值  |
|---------- |-------------- |--------------------------------  | ------|
| closeIcon | 可传文字或img标签更改弹窗关闭按钮样式 |  文字或img标签 | x |
| content | 内容 |  String | 内容 |
