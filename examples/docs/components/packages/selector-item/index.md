## SelectorItem 选择器子组件

### 概述

用于 `CityPicker`  与 `CarPicker` 中自定义选项列表

### 代码演示

::: demo selector-item
选择器子组件
:::

### API

#### SelectorItem 参数
| 参数  | 说明                                   | 类型   | 可选值 | 是否必填 | 默认值 |
|-------|---------------------------------------|--------|--------|-------|--------|
| title | 这部分内容的title                       | string | —     | 否     | 无     |
| index | 子组件内容的索引，传值会对应显示在右边导航栏 | string | —     | 否     | 无     |

#### SelectorItem slot
| slot名称 | 说明                             |
| -------- | -------------------------------- |
| default       | 子组件自定义内容 |
