range

### API

#### Range
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| range | 初始范围(需要还原到初始范围时，给value赋值range) | string | — | — |
| value | 当前游标范围 | string | — |  |
| step  | 选择间隔 | number | — | — |
| tipList | 下标提示 | array | — | — |
| primary-color | 下标提示的字体颜色 | string | — | — |
| range-color | 范围条颜色 | string | — | — |
| cursor-img | 游标图像 | string | — |  |
